# Python Notes Manager

A simple notes manager written in Python.

## Features

- Add notes
- View notes
- Save notes to a text file
- Load notes from a text file
- Input validation
- Console application

## Technologies

- Python 3

## How to Run

```bash
python notes_manager.py
```

## Example

```text
--- NOTES MANAGER ---
1. Add note
2. View notes
3. Save notes
4. Load notes
5. Exit
```
